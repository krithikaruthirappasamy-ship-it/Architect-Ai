 Architect AI

Architect AI is a modern AI-powered web application that generates house layouts and architectural designs based on user input. It features a sleek dark theme with orange accents and provides an interactive interface for designing and visualizing house layouts.


 Features
 AI-based house layout generator
 Modern dark + orange UI theme
 Intelligent layout-based design suggestions
 Responsive web design (desktop & mobile friendly)
 Sections for Home, AI Generator, Contact, Architect Details, and Login
 Interactive layout selection and design preview

Technologies Used

- HTML5
- CSS3
- JavaScript
